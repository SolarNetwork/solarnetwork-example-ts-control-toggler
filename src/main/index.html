SolarNetwork Developer Example: TypeScript Control Toggler

This is an example app for developers that demonstrates how to use the SolarNetwork API with the help of the solarnetwork-control-toggler JavaScript library to make authenticated API calls to monitor and update a SolarNode control value.

Fill in the details in the form and click the Start button to start monitoring the control. Once started, use the Update form to change the value of the control. A spinning indicator will show while the change is still pending.

See the project on Github for more information.
